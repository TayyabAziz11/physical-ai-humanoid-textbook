# ماڈیول 1: ROS 2 – روبوٹک اعصابی نظام

## جائزہ

ROS 2 (Robot Operating System 2) وہ مڈل ویئر ہے جو روبوٹ کے مختلف حصوں کے درمیان رابطے کو طاقت دیتا ہے۔ اسے ایک اعصابی نظام کے طور پر سوچیں جو سینسرز، actuators، اور AI دماغوں کو ایک دوسرے سے بات کرنے کی اجازت دیتا ہے۔

## آپ اس ماڈیول میں کیا سیکھیں گے

- ROS 2 کی تعمیر اور بنیادی تصورات (nodes، topics، services)
- Python (`rclpy`) کے ساتھ ROS 2 پیکجز بنانا
- ہیومنائیڈز کے لیے URDF (Unified Robot Description Format) کو سمجھنا
- Python AI ایجنٹس کو ROS کنٹرولرز سے جوڑنا
