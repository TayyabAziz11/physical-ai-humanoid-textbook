# Module 4: Vision-Language-Action (VLA)

## جائزہ

VLA ماڈلز computer vision، natural language understanding، اور robotic control کو یکجا کرتے ہیں تاکہ robots پیچیدہ ہدایات کو سمجھ کر ان پر عمل کر سکیں۔

## آپ کیا سیکھیں گے

- VLA ماڈل کی تعمیرات
- زبان کی ہدایات کے ساتھ robots کی تربیت
- Manipulation کے لیے vision اور language کو integrate کرنا
