# Module 2: Digital Twin (Gazebo اور Unity)

## جائزہ

Digital twins ہمیں حقیقی دنیا میں تعینات کرنے سے پہلے virtual ماحول میں robots کو simulate کرنے کی اجازت دیتے ہیں۔ اس module میں Gazebo (physics simulation) اور Unity (high-fidelity rendering) شامل ہیں۔

## آپ کیا سیکھیں گے

- Robotics simulation کے لیے Gazebo کو ترتیب دینا
- Humanoid robots کے لیے URDF ماڈلز بنانا
- حقیقت پسندانہ rendering کے لیے Unity کو integrate کرنا
- Sim-to-real transfer تکنیکیں
