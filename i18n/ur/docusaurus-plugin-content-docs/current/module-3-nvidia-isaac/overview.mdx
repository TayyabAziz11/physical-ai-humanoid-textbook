# Module 3: NVIDIA Isaac (AI-Robot Brain)

## جائزہ

NVIDIA Isaac robotics کے لیے جدید AI صلاحیتیں فراہم کرتا ہے، بشمول deep learning کے ذریعے perception، navigation، اور manipulation۔

## آپ کیا سیکھیں گے

- Robot تربیت کے لیے Isaac Sim
- Isaac SDK کے ساتھ Perception
- AI سے چلنے والا navigation اور manipulation
