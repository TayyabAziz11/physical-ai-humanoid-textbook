# مبارک ہو!

آپ نے ابھی **Docusaurus کی بنیادیں** سیکھ لی ہیں اور **ابتدائی template** میں کچھ تبدیلیاں کی ہیں۔

Docusaurus کے پاس **بہت کچھ اور پیش کرنے کے لیے** ہے!

کیا آپ کے پاس **5 منٹ اور** ہیں؟ **[versioning](../tutorial-extras/manage-docs-versions.md)** اور **[i18n](../tutorial-extras/translate-your-site.md)** پر ایک نظر ڈالیں۔

اس tutorial میں کوئی بات **غیر واضح** یا **buggy** ہے؟ [براہ کرم اس کی اطلاع دیں!](https://github.com/facebook/docusaurus/discussions/4610)

## اب کیا ہے؟

- [سرکاری دستاویزات](https://docusaurus.io/) پڑھیں
- اپنی سائٹ کی تشکیل [`docusaurus.config.js`](https://docusaurus.io/docs/api/docusaurus-config) سے ترمیم کریں
- [`themeConfig`](https://docusaurus.io/docs/api/themes/configuration) کے ساتھ navbar اور footer items شامل کریں
- ایک custom [Design اور Layout](https://docusaurus.io/docs/styling-layout) شامل کریں
- ایک [search bar](https://docusaurus.io/docs/search) شامل کریں
- [Docusaurus showcase](https://docusaurus.io/showcase) میں الہام تلاش کریں
- [Docusaurus Community](https://docusaurus.io/community/support) میں شامل ہوں
