# کثیر لسانی ترجمے کی رہنما

کثیر لسانی ترجمے کی خصوصیت میں خوش آمدید! یہ رہنما آپ کو اس textbook میں کسی بھی مواد کو اپنی پسندیدہ زبان میں ترجمہ کرنے میں مدد کرے گی۔

## جائزہ

ترجمے کی خصوصیت آپ کو اجازت دیتی ہے:
- کسی بھی صفحے سے منتخب متن کو 7 معاون زبانوں میں ترجمہ کرنا
- Chatbot کے جوابات کو اپنی پسندیدہ زبان میں ترجمہ کرنا
- عربی اور اردو کے لیے مناسب right-to-left (RTL) سپورٹ کے ساتھ ترجمے دیکھنا
- تیز تر بار بار رسائی کے لیے خودکار طور پر ترجموں کو cache کرنا

تمام ترجمے OpenAI GPT-4o-mini کے ذریعے فراہم کیے جاتے ہیں اور درستگی کے لیے تکنیکی اصطلاحات کو انگریزی میں محفوظ رکھتے ہیں۔

:::

## منتخب متن کا ترجمہ کیسے کریں

### مرحلہ 1: متن منتخب کریں

اپنے mouse یا keyboard کا استعمال کرتے ہوئے صفحے پر کوئی بھی متن منتخب کریں۔ متن ہو سکتا ہے:
- ایک واحد لفظ
- ایک جملہ یا پیراگراف
- 1,500 حروف تک

:::info انتخاب کی حدود
1,500 حروف سے زیادہ لمبے انتخابات کو مسترد کر دیا جائے گا۔ طویل مواد کے لیے، چھوٹے حصوں میں ترجمہ کریں۔
:::

### مرحلہ 2: اپنی زبان منتخب کریں

جب آپ متن منتخب کرتے ہیں، آپ کے انتخاب کے قریب ایک ترجمے کا panel ظاہر ہوگا جس میں:
- تمام 7 معاون زبانوں کو دکھانے والا **زبان کا selector dropdown**
- ایک **Translate بٹن**

زبان کے dropdown پر کلک کریں اور اپنی پسندیدہ ہدف زبان منتخب کریں۔

### مرحلہ 3: ترجمہ کریں

**Translate** بٹن پر کلک کریں۔ ترجمہ پروسیس ہوتے وقت آپ کو loading indicator نظر آئے گا۔

### مرحلہ 4: ترجمہ دیکھیں

ایک modal window ظاہر ہوگی جس میں:
- **اصل متن**: وہ متن جو آپ نے منتخب کیا (اگر چاہیں تو چھپایا جا سکتا ہے)
- **ترجمہ شدہ متن**: آپ کی منتخب کردہ زبان میں ترجمہ
- **زبان کی معلومات**: flags کے ساتھ source اور target زبانیں
- **Cache کی حیثیت**: کیا ترجمہ cache سے لوڈ ہوا (⚡ Cached بیج)

### مرحلہ 5: کاپی کریں یا بند کریں

آپ کر سکتے ہیں:
- 📋 Copy بٹن استعمال کرتے ہوئے ترجمہ شدہ متن کو اپنے clipboard میں **کاپی** کریں
- Show/Hide بٹن سے اصل متن کی visibility کو **Toggle** کریں
- X بٹن پر کلک کرکے، باہر کلک کرکے، یا Escape دباکر modal کو **بند** کریں

## ترجمے کی خصوصیات

### تکنیکی اصطلاحات کا تحفظ

بطور ڈیفالٹ، تکنیکی اصطلاحات درستگی کو یقینی بنانے کے لیے انگریزی میں رہتی ہیں۔ مثال کے طور پر:
- "ROS 2" تمام زبانوں میں "ROS 2" رہتا ہے
- "SLAM" "SLAM" رہتا ہے
- "Docker" "Docker" رہتا ہے

یہ یقینی بناتا ہے کہ تکنیکی documentation واضح اور قابل تلاش رہے۔

### Client-Side Caching

ترجمے خودکار طور پر 7 دنوں کے لیے آپ کے browser میں cache ہوتے ہیں۔ اس کا مطلب ہے:
- ✅ **تیز**: بار بار ترجمے فوری طور پر لوڈ ہوتے ہیں
- ✅ **رازداری**: ترجمے صرف آپ کے device پر محفوظ ہوتے ہیں
- ✅ **آف لائن رسائی**: Cached ترجمے internet کے بغیر کام کرتے ہیں
- ✅ **Server پر کوئی بوجھ نہیں**: API calls اور اخراجات کم کرتا ہے

Cache خود کو خودکار طور پر منظم کرتا ہے:
- پرانے ترجمے 7 دنوں کے بعد ختم ہو جاتے ہیں
- ہر زبان کے لیے زیادہ سے زیادہ 50 ترجمے
- جب cache بھر جائے تو کم سے کم حال ہی میں استعمال شدہ ترجمے ہٹا دیے جاتے ہیں

### Right-to-Left (RTL) سپورٹ

عربی اور اردو کے ترجمے خودکار طور پر مناسب دائیں سے بائیں متن کی سمت کے ساتھ ظاہر ہوتے ہیں:
- متن دائیں سے بائیں بہتا ہے
- رموز اوقاف صحیح طریقے سے ظاہر ہوتے ہیں
- ملا جلا مواد (عربی/اردو میں انگریزی اصطلاحات) صحیح طریقے سے render ہوتا ہے

## عام منظرنامے

### Code مثال کا ترجمہ کرنا

جب ایسے مواد کا ترجمہ کر رہے ہوں جس میں code کی مثالیں شامل ہوں:
1. صرف متن کی تفصیل منتخب کریں، خود code نہیں
2. Code blocks اور تکنیکی اصطلاحات اپنی اصل شکل میں رہتی ہیں
3. Comments کو الگ سے منتخب کرکے ترجمہ کیا جا سکتا ہے

### متعدد حصوں کا ترجمہ کرنا

متعدد پیراگرافوں کا ترجمہ کرنے کے لیے:
1. ایک وقت میں ایک حصے کا ترجمہ کریں (زیادہ سے زیادہ 1,500 حروف)
2. ہر ترجمے کو الگ سے کاپی اور محفوظ کیا جا سکتا ہے
3. تمام ترجمے بعد میں تیز رسائی کے لیے cache ہوتے ہیں

### زبانیں تبدیل کرنا

ایک ہی متن کو متعدد زبانوں میں دیکھنے کے لیے:
1. Modal بند کرنے کے بعد متن دوبارہ منتخب کریں
2. ایک مختلف ہدف زبان منتخب کریں
3. دوبارہ ترجمہ کریں (cached ترجمے فوری طور پر لوڈ ہوتے ہیں)

## مسائل کا حل

### "Rate limit exceeded" خرابی

**مسئلہ**: آپ نے کم وقت میں بہت زیادہ ترجمے کی درخواستیں کی ہیں۔

**حل**: دوبارہ ترجمہ کرنے سے پہلے 1 منٹ انتظار کریں۔ حد ہر صارف کے لیے 10 ترجمے فی منٹ ہے۔

### "Selection too long" خرابی

**مسئلہ**: منتخب متن 1,500 حروف سے زیادہ ہے۔

**حل**: متن کا ایک چھوٹا حصہ منتخب کریں اور اسے حصوں میں ترجمہ کریں۔

### "Translation failed" خرابی

**ممکنہ وجوہات**:
- عارضی network کا مسئلہ
- Backend service دستیاب نہیں
- غلط متن کا انتخاب

**حل**:
1. اپنا internet connection چیک کریں
2. متن کو دوبارہ منتخب کرنے کی کوشش کریں
3. صفحہ refresh کریں اور دوبارہ کوشش کریں
4. اگر مسئلہ برقرار رہے، کچھ منٹ انتظار کریں

### ترجمہ ظاہر نہیں ہو رہا

**ممکنہ وجوہات**:
- متن کا انتخاب بہت چھوٹا تھا یا صرف whitespace پر مشتمل تھا
- آپ کے browser میں JavaScript غیر فعال ہے
- Ad blocker یا privacy extension خصوصیت کو روک رہی ہے

**حل**:
1. یقینی بنائیں کہ آپ نے اصل متن منتخب کیا ہے (صرف spaces نہیں)
2. JavaScript فعال کریں
3. عارضی طور پر extensions غیر فعال کریں اور دوبارہ کوشش کریں

## رازداری اور ڈیٹا

### ہم کیا محفوظ کرتے ہیں

- **آپ کے device پر**: localStorage میں cache شدہ ترجمہ شدہ متن (7 دنوں تک)
- **ہمارے servers پر**: کوئی صارف کا ڈیٹا یا ترجمے مستقل طور پر محفوظ نہیں ہوتے

### ہم کیا محفوظ نہیں کرتے

- ❌ اصل کتاب کا مواد
- ❌ صارف کے انتخابات
- ❌ مقامی cache سے آگے ترجمے کی تاریخ
- ❌ ذاتی معلومات

### ڈیٹا کی منتقلی

جب آپ متن کا ترجمہ کرتے ہیں:
1. منتخب متن ہمارے backend API کو بھیجا جاتا ہے
2. Backend ترجمے کے لیے OpenAI کے API کو call کرتا ہے
3. ترجمہ آپ کے browser کو واپس کیا جاتا ہے
4. Request اور response دونوں عارضی ہیں اور logged نہیں ہوتے

:::info رازداری پر مبنی ڈیزائن
ہم آپ کے انتخابات یا ترجموں کو اپنے servers پر کبھی محفوظ نہیں کرتے۔ تمام caching آپ کے browser میں ہوتی ہے۔
:::

## بہترین نتائج کے لیے تجاویز

### 1. مکمل جملے منتخب کریں

بہتر ترجمے کے معیار کے لیے، الگ الفاظ کی بجائے مکمل جملے یا پیراگراف منتخب کریں۔

### 2. سیاق و سباق فراہم کریں

ترجمے کا نظام قدرتی زبان کے ساتھ بہترین کام کرتا ہے۔ مکمل خیالات کا ٹکڑوں سے زیادہ درست ترجمہ ہوتا ہے۔

### 3. تکنیکی مواد

انتہائی تکنیکی مواد کے لیے، تکنیکی اصطلاحات کا حوالہ دینے کے لیے انگریزی version کو visible رکھیں۔

### 4. طویل دستاویزات

پورے ابواب کے ترجمے کے لیے:
- اپنی زبان میں سوالات پوچھنے کے لیے chatbot استعمال کریں
- ضرورت کے مطابق کلیدی حصوں کا ترجمہ کریں
- اہم ترجموں کے ساتھ ایک حوالہ document رکھیں

## اکثر پوچھے جانے والے سوالات

### کیا میں پوری textbook کا ترجمہ کر سکتا ہوں؟

فی الوقت، آپ ایک وقت میں 1,500 حروف تک کے منتخب حصوں کا ترجمہ کر سکتے ہیں۔ مکمل باب کے ترجموں کے لیے، chatbot استعمال کریں اور اپنی پسندیدہ زبان میں سوالات پوچھیں۔

### کیا ترجمے machine سے بنائے گئے ہیں یا انسانوں نے جائزہ لیا ہے؟

تمام ترجمے OpenAI GPT-4o-mini کے ذریعے بنائے جاتے ہیں۔ اگرچہ عام طور پر درست ہیں، machine کے ترجمے کبھی کبھار نزاکتوں کو miss کر سکتے ہیں۔ اہم تکنیکی تفصیلات کے لیے ہمیشہ انگریزی version کا حوالہ دیں۔

### کیا میں نئی زبان کی تجویز دے سکتا ہوں؟

زبان کی سپورٹ فی الوقت اوپر دی گئی 7 زبانوں تک محدود ہے۔ خصوصیت کی درخواستیں GitHub issues کے ذریعے جمع کرائی جا سکتی ہیں۔

### کیا میری زبان کی ترجیح یاد رکھی جائے گی؟

جی ہاں! آپ کی آخری بار منتخب کردہ ہدف زبان آپ کے browser میں محفوظ ہو جاتی ہے اور اگلی بار جب آپ ترجمہ کریں گے تو پہلے سے منتخب ہوگی۔

### کیا ترجمہ آف لائن کام کرتا ہے؟

Cached ترجمے آف لائن کام کرتے ہیں۔ نئے ترجموں کے لیے translation API سے رابطہ کرنے کے لیے internet connection کی ضرورت ہوتی ہے۔

### ترجمے کتنے درست ہیں؟

ترجمے OpenAI کے GPT-4o-mini ماڈل کے ذریعے فراہم کیے جاتے ہیں، جو عام اور تکنیکی مواد کے لیے اعلیٰ معیار کے نتائج فراہم کرتا ہے۔ تکنیکی اصطلاحات درستگی برقرار رکھنے کے لیے انگریزی میں محفوظ رہتی ہیں۔

## مدد حاصل کرنا

اگر آپ کو ایسے مسائل کا سامنا ہو جو اس رہنما میں شامل نہیں:

1. **اوپر FAQ چیک کریں**
2. **صفحہ Refresh کریں** اور دوبارہ کوشش کریں
3. اگر ترجمے غلط لگیں تو **اپنا browser cache صاف کریں**
4. ہماری GitHub repository پر **ایک issue رپورٹ کریں**
